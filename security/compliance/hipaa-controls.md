# HIPAA Controls (if applicable)

**Purpose**: Healthcare data protection  
**Description**: Controls for Protected Health Information (PHI)

**Note**: Only applicable if handling health-related products/data

## Administrative Safeguards
- [ ] Security management process
- [ ] Assigned security responsibility
- [ ] Workforce security procedures
- [ ] Information access management

## Physical Safeguards
- [ ] Facility access controls
- [ ] Workstation use policies
- [ ] Device and media controls

## Technical Safeguards
- [ ] Access control with unique user IDs
- [ ] Audit controls logging all access
- [ ] Integrity controls for data authentication
- [ ] Transmission security with encryption

## Business Associate Agreements
- [ ] BAAs with all PHI processors
- [ ] Subcontractor management

## Breach Notification
60-day notification requirement to affected individuals

**Implementation**: Only if selling healthcare products
